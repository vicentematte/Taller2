# Taller 2 - Simulacion Pokemon

## Descripcion del proyecto
Este taller consistio en crear un juego de simulacion de Pokemon por consola usando aplicando conocimineto de POO. El juego permite al jugador capturar Pokemon, armar su equipo, batallar contra lideres de gimnasios, para poder lograr diversos objetivos de acuerdo al juego.

## Como se realizo:
El taller numero dos se ralizo primeramente abordando todas las funciones para poder cargar los objetos en sus respectivas clases, luego se fueron desglosando las opciones que nos pedian para asi ir avanzando y lograr finalizar lo solicitado. Cada parte de nuestro menu se logro hacer funcionar luego de varias horas de estudio autonomo, aprendiendo asi a utilizar las funciones de random, o escritura de archivos. 

## Clases utilizadas
- Main.java - logica principal, menus y desarrollo del juego
- Pokedex.java - Pokemon con todos sus stats (vida, ataque, defensa, etc)
- Gimnasio.java - Gimnasio con su lider y sus Pokemon
- altoMando.java - Miembro del Alto Mando y sus Pokemon
- Habitat.java - Zona donde se pueden capturar Pokemon
- TablaTipos.java - Matriz de efectividad de tipos para los combates

## Instrucciones de ejecucion
1. Abrir el proyecto en Eclipse
2. Asegurarse de que los archivos .txt esten dentro del proyecto para que funcione
3. Ejecutar Main.java y disfrutar de esta increible simulacion
