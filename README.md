## Descripcion del proyecto
Juego de simulacion de Pokemon por consola hecho en Java. Permite capturar Pokemon, batallar contra lideres de gimnasio y el Alto Mando, gestionar el equipo y guardar la partida.

## Instrucciones de ejecucion
1. Abrir el proyecto en Eclipse
2. Asegurarse de que los archivos .txt esten en la raiz del proyecto
3. Ejecutar `Main.java`
